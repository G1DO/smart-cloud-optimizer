# AWS Usage Data (CSV)

المشروع ده فيه داتا متجمعة من AWS عن:
- EC2 Instances
- Metrics (CPU / Network / Disk)
- Daily AWS Costs

## محتوى الملفات

### 1) ec2_instances.csv
بيانات كل الـ EC2 Instances:
- Instance ID  
- Region  
- Instance Type  
- Tags  
- State  

### 2) ec2_metrics.csv
Metrics لكل Instance:
- CPU Average / Max  
- NetworkIn / NetworkOut  
- Disk Read / Write  
- Timestamp  

### 3) daily_cost_by_service.csv
تكلفة كل Service يوم بيوم.

## طريقة العرض
تقدر تفتح كل CSV باستخدام:
- Excel  
- Google Sheets  
- أي برنامج CSV Viewer
